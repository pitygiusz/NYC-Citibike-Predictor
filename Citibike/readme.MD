# NYC Citibike

This part of the NYC-ML-Project focuses on exploring and analyzing the NYC Citibike dataset.  
The goal was to gain insights into bike usage patterns in New York City and build a models to predict its usage in the future, which could be helpful to plan bike relocations or new stations locations.
