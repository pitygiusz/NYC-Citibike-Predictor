# NYC Parking Fines

This part of the NYC-ML-Project focuses on the **NYC Parking Fines** dataset. 

The goal was to analyze NYC parking violations data (Fiscal Year 2015) to understand violation patterns and predict fine categories based on vehicle and violation characteristics.
