# NYC Parking Fines

This part of the **NYC-ML-Project** focuses on the **NYC Parking Fines** dataset. 

## Goal
The goal was to analyze NYC parking violations data to understand violation patterns and predict fine categories based on vehicle and violation characteristics.

## Technologies Used
- Python (Pandas, NumPy, Scikit-Learn , XGBoost)
- Matplotlib / Seaborn
- Jupyter Notebook

## Dataset
The dataset used is provided publicly by City of New York and is available at [NYC OpenData](https://data.cityofnewyork.us/City-Government/Parking-Violations-Issued-Fiscal-Year-2023/869v-vr48/about_data) website. My analysis refers to fiscal year 2015.

## Project Structure
1. Loading data
2. Basic preprocessing
3. Exploratory data analisys
4. Hypothesis testing
5. Predicting Traffic Fine Amounts
